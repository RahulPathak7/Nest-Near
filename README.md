NestNear – Smart Nearby Accommodation Finder
📌 Overview

NestNear is a location-based web application that helps users find nearby accommodations such as PGs, hostels, and rental rooms quickly and efficiently. It is designed to simplify the house-hunting process for students and working professionals moving to new cities.

🚀 Features
📍 Location-based accommodation search
🔎 Smart filtering (price, distance, type)
🏠 Detailed listings with images and information
📱 Responsive design (mobile + desktop)
⚡ Fast and user-friendly interface
